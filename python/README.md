# dependencies
this bot uses discord.py
* ## discord.py
  [git](https://github.com/Rapptz/discord.py)
  ```sh
      # Linux/macOS
      python3 -m pip install -U discord.py

      # Windows
      py -3 -m pip install -U discord.py
  ```
* ## jishaku
  [git](https://github.com/Gorialis/jishaku)
  ```
      python3 -m pip install -U jishaku
  ```
